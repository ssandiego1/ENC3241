Step-by-Step Guide

How to Create a Folder Structure to Organize Your Files

📁 Beginner-Friendly 🖥️ For Windows ⏱ ~15 Minutes

Step-by-Step Instructions

  1. 1

    Open File Explorer

    Press the yellow folder icon on your taskbar at the bottom of the screen, or press Windows key + E on your keyboard. File Explorer will open, showing your Quick Access folders.

    📌 This is your file manager — think of it as the map of your computer.

Figure 1. File Explorer open on Windows, showing Quick Access folders. (Source: Samuel San Diego)

  1. 2

    Click on "Documents" in the left sidebar

    In the panel on the left side of File Explorer, click the word Documents. This opens your Documents folder — a safe, easy-to-find place to build your new folder system.

  2. 3

    Right-click on an empty area inside the Documents folder

    Move your mouse to an empty white space in the main area (not on any existing file or folder) and press the right mouse button. A menu will pop up with several options.

  3. 4

    Hover over "New" in the menu, then click "Folder"

    In the right-click menu, move your mouse over the word New. A second small menu will slide out. Click Folder from that second menu. A new folder will appear with its name highlighted and ready to edit.

    📌 The name will be highlighted in blue — that means it's ready for you to type!

Figure 2. Right-clicking inside a folder reveals the "New > Folder" option. (Source: Samuel San Diego)

  1. 5

    Type the name "School" and press Enter

    While the folder name is highlighted, type School on your keyboard, then press Enter to save the name. The folder is now created.

    ⚠️ Don't click away before pressing Enter — you'll accidentally keep the name "New Folder"!
  2. 6

    Repeat steps 3–5 to create a "Personal" folder

    Right-click in the empty space again, select New → Folder, type Personal, and press Enter. You now have two organized category folders side by side.

  3. 7

    Repeat steps 3–5 one more time to create a "Work" folder

    Right-click again, select New → Folder, type Work, and press Enter. You now have three main category folders: School, Personal, and Work.

  4. 8

    Double-click the "School" folder to open it, then create a subfolder for a semester

    Double-click School to enter it. Then right-click → New → Folder and name it Fall 2024. This gives each semester its own section inside School.

    💡 You can keep going deeper — e.g., a folder for each class inside Fall 2024.
  5. 9

    Move an existing file into your new folder by dragging and dropping it

    Find a file you already have (such as in Downloads). Click and hold it with your mouse, drag it over to the correct folder in your new structure, and release. The file now lives in its proper home.

    💡 Alternative: right-click the file → Cut → open the destination folder → right-click → Paste.

Figure 3. The completed folder structure with School, Personal, and Work folders visible. (Source: Samuel San Diego)

✅ Pro tip: Once your structure is set up, save new files directly into the right folder from the start — instead of dumping everything on the Desktop and sorting later. This takes 2 extra seconds now and saves 20 minutes of frustrated searching later.

You Did It! 🎉

You've built a folder structure that will keep your computer organized. You now have three main category folders (School, Personal, Work), a semester-based subfolder inside School, and you know how to move and rename files. This same system scales up as your files grow — just add new subfolders whenever you need them.
